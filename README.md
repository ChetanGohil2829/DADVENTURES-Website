## Setup
1. Unzip `DADVENTURES-v1.0.1.zip`.
2. Open `index.html` to see the Welcome screen (30s auto-redirect to Home).
3. For Events/Shop/Blog JSON to load locally, serve via a local server (e.g. `npx serve .`).
